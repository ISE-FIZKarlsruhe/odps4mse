# Contributing

We welcome contributions to this collection of Ontology Design Patterns for Materials Science and Engineering.

Please see the full [Contributing Guide](https://ise-fizkarlsruhe.github.io/odps4mse/contributing/) on our documentation site for detailed instructions on:

- Adding new patterns
- Adding new ontologies
- Code style and CI checks
- Reporting issues

## Quick Start

1. Fork the repository and clone it locally.
2. Create a new branch: `git checkout -b my-pattern`
3. Add your pattern using the [template](Patterns/template.md).
4. Validate your files: `robot report -i your-pattern.ttl --fail-on ERROR`
5. Push and open a Pull Request.
